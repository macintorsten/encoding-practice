# Encoding practice

## What is it?

A lab meant to teach how and when to apply which type of encoding to avoid reflective XSS bugs

## How to start?

1. Install PHP
2. Run command: `php -S 127.0.0.1:8000`
3. Open you browser and navigate to [http://127.0.0.1:8000](http://127.0.0.1:8000) (Firefox recommended)
